The Busy Bean Database

By Reed Hardin & Kazu Ishihara

Menu of Operations

  • Add/Remove Customers

    • INSERT or DELETE Customer from Customers table


  • Add/Remove Products

    • INSERT or DELETE Product from Products table


  • Add/Remove Orders

    • INSERT or DELETE Order from Orders table


  • Add/Remove Beans

    • INSERT or DELETE Bean from Beans table


  • Add/Remove Suppliers

    • INSERT or DELETE Supplier from Suppliers table


  • Add/Remove Product Details

    • INSERT or DELETE product details to Products_Beans table (M:M)


  • Add/Remove Order Details

    • INSERT or DELETE order details to Orders_Products table (M:M)


  • Display customer/bean names from orders

    • SELECT customer names and bean types from Orders


  • Display customer order details

    • SELECT customer names, order numbers, products, product quantities and suppliers


  • Display customer bean preferences

    • SELECT customer names based off bean type to see how popular a particular bean type is

    Ability to search which customers prefer a certain type of bean with a bean type filter


  • Update Suppliers

    • UPDATE the Suppliers table
